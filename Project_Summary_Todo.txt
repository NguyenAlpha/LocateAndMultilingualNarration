TÓM TẮT ĐỒ ÁN: LOCATE AND MULTILINGUAL NARRATION
Hệ thống Bản đồ & Thuyết minh Đa ngôn ngữ cho Phố Ẩm Thực
========================================================================

1. MỤC TIÊU ĐỒ ÁN
------------------------------------------------------------------------
Xây dựng ứng dụng di động hỗ trợ du khách tham quan Phố Ẩm Thực một cách thông minh:
- Định vị và hiển thị gian hàng trên bản đồ tương tác.
- Nghe thuyết minh audio đa ngôn ngữ tự động hoặc thủ công.
- Cá nhân hóa trải nghiệm theo thiết bị (DevicePreferences - không cần đăng nhập).
- Hỗ trợ quét QR để nhanh chóng focus gian hàng.

Luồng chính: MainPage → Quét QR → Chọn ngôn ngữ & giọng đọc → Bản đồ + Nghe thuyết minh.

2. KIẾN TRÚC & CÔNG NGHỆ
------------------------------------------------------------------------
- Mobile App     : .NET MAUI 10.0 (MVVM + Dependency Injection)
- Bản đồ         : Mapsui + OpenStreetMap
- Quét QR        : ZXing.Net.Maui
- Âm thanh       : Plugin.Maui.Audio
- Cache local    : SQLite (LocalStallRepository + AudioCacheService)
- Backend        : ASP.NET Core 10.0 Web API + EF Core
- Database       : SQL Server
- TTS            : Azure Cognitive Services Speech (qua backend)
- Storage        : Azure Blob Storage

3. CHỨC NĂNG ĐÃ HOÀN THÀNH
------------------------------------------------------------------------
[✓] MainPage với Quick Actions (Bản đồ, Ngôn ngữ, Âm thanh, Hồ sơ)
[✓] Quét QR Code → điều hướng đến chọn ngôn ngữ → Map

[✓] Cache-First + SQLite cho stalls
[✓] Navigation guard & xử lý một số lỗi cơ bản

4. CƠ SỞ DỮ LIỆU & DEVICE PREFERENCES
------------------------------------------------------------------------
Sử dụng bảng DevicePreferences để lưu cấu hình theo thiết bị (anonymous):
- DeviceId (unique)
- LanguageId / LanguageCode
- Voice (TtsVoiceProfileId)
- SpeechRate
- AutoPlay
- Platform, DeviceModel, OsVersion, FirstSeenAt, LastSeenAt

5. API ĐANG SỬ DỤNG TRONG MOBILE
------------------------------------------------------------------------
- GET  /api/geo/stalls?deviceId=...
- GET  /api/languages/active
- GET  /api/tts-voice-profiles/active?languageId=...
- POST /api/device-preference
- GET  /api/device-preference/{deviceId}
- GET  /api/geo/nearest-stall
